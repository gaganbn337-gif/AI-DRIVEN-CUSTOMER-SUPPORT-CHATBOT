# AI Chatbot - ChatGPT Clone

A professional AI chatbot interface similar to ChatGPT with user authentication, sidebar navigation, voice assistance, file upload, and persistent chat history.

## 🎯 Two Modes Available:

### 1️⃣ Basic Mode (Current - No Setup)
- Works immediately without any setup
- Built-in intelligent responses
- Good for basic questions (time, date, simple info)

### 2️⃣ AI Mode (Recommended - 2 Minutes Setup)
- **REAL AI like ChatGPT/Gemini**
- Answers ANY question with detailed information
- **100% FREE with Google Gemini API**
- See `API_SETUP_GUIDE.md` for instructions

---

## 🚀 Quick Start

### Option A: Use Without AI (Works Now)

## 📖 Setup Instructions

### Step 1: Get FREE Gemini API Key (2 Minutes)

1. Go to: **https://aistudio.google.com/app/apikey**
2. Sign in with Google
3. Click "Get API Key" → "Create API key in new project"
4. Copy the API key (starts with: AIzaSy...)

### Step 2: Add API Key to Chatbot

1. Open `script.js` file
2. Find line 350:
   ```javascript
   const GEMINI_API_KEY = 'YOUR_API_KEY_HERE';
   ```
3. Replace `YOUR_API_KEY_HERE` with your actual key:
   ```javascript
   const GEMINI_API_KEY = 'AIzaSyXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX';
   ```
4. Save the file

### Step 3: Test It!

1. Open `index.html` in browser
2. Login to your account
3. Ask: "Explain artificial intelligence"
4. Ask: "Write a short poem about coding"
5. Enjoy ChatGPT-level responses! 🎉

**For detailed instructions, see `API_SETUP_GUIDE.md`**

## 🎓 How to Use

### First Time Setup
1. Open `index.html` in your browser
2. Enter a username and password
3. Click "Login / Sign Up"
4. Start chatting!

### Basic Features
- **Type Messages** - Type and press Enter or click Send
- **Voice Input** - Click 🎤 to speak (Chrome/Edge recommended)
- **Upload Files** - Click 📎 to upload files
- **Toggle Voice** - Click 🔊/🔇 for voice responses

### Chat Management
- **New Chat** - Click "✏️ New chat" for fresh conversation
- **Switch Chats** - Click any chat in sidebar to load it
- **Logout** - Click 🚪 icon to logout
