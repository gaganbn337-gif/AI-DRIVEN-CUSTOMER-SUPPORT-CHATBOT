# 🔧 How to Fix Gemini API Error

## The Problem
You're getting error: "models/gemini-pro is not found for API version v1"

This means the Gemini API is not enabled in your Google Cloud project.

---

## ✅ SOLUTION 1: Enable Gemini API (5 Minutes)

### Step 1: Go to Google Cloud Console
1. Visit: https://console.cloud.google.com/
2. Sign in with the same Google account you used for the API key

### Step 2: Enable the Generative Language API
1. Go to: https://console.cloud.google.com/apis/library/generativelanguage.googleapis.com
2. Click "ENABLE" button
3. Wait for it to enable (takes 1-2 minutes)

### Step 3: Test Again
1. Refresh your chatbot page
2. Try asking a question
3. Should work now! ✅
